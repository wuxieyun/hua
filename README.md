# 你画我猜辅助工具

一款基于 Go 语言开发的 Windows 桌面应用，专为"你画我猜"类游戏设计的自动化绘制工具。

## 功能特性

- **图片解析**：支持 JPEG、PNG、GIF、WebP 等多种图片格式的加载与解析
- **边缘检测**：通过模糊、阈值等参数自动提取图片轮廓路径
- **路径优化**：提供路径简化功能，减少冗余绘制点
- **画布校准**：支持自定义绘制区域的选择与校准
- **参数调整**：可调节绘制速度、线条间距、模糊度等多项参数
- **实时预览**：提供图片处理效果的预览功能
- **自动绘制**：通过模拟鼠标点击自动在游戏画布上完成绘制

## 技术实现

- 使用 Go 语言结合 Windows API 构建原生图形界面
- 采用图像处理算法实现轮廓提取与路径生成
- 通过 `user32.dll` 和 `gdi32.dll` 实现鼠标操作与窗口管理
- 支持多线程异步处理，保证界面响应流畅

## 使用场景

适用于各类在线"你画我猜"游戏，帮助用户快速将参考图片转化为游戏画面中的绘制内容。
